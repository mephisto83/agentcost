# AgentCost

AI cost optimization proxy for OpenAI/Anthropic API calls.

## Features
- Budget enforcement per session, per hour, per request
- - Utility scoring with configurable thresholds
  - - Semantic deduplication to avoid redundant API calls
    - - Real-time analytics dashboard
      - - React + Tailwind SaaS webapp with Firebase Auth
       
        - ## Quick Start
        - 1. Clone this repo
          2. 2. Install: `pip install -e .`
             3. 3. Run: `agentcost --help`
               
                4. ## Web App
                5. The `webapp/` directory contains a full SaaS frontend built with React + Tailwind CSS.
